# sql-join-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What is a foreign key?
  A foreign key is a key used to link two tables together. It is a field (or collection of fields) in one table that uniquely identifies a row of another table or the same table.

- How do you join two SQL tables? (Provide at least two syntaxes.)
  "join" "table" "using" "("column)"

- How do you temporarily rename columns or tables in a SQL statement?
  Using the "as" keyword.

- How do you create a one-to-many relationship between two tables?
  Using a foreign key

- How do you create a many-to-many relationship between two tables?
  By creating a third table linking two together.
