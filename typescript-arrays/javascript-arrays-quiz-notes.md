# javascript-arrays-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What are arrays used for?
  To store ordered collections.

- Describe array literal notation.
  Brackets notating an array assigned to a variable. const arr = [];

- How are arrays different from "plain" objects?
  Objects allow you to store keyed collections of unordered values.
  Whereas arrays are ordered and don't use keys. They allow insertion of values at specified indexes.

- What number represents the first index of an array?
  [0].

- What is the `length` property of an array?
  The total number of values inside an array.

- How do you calculate the last index of an array?
  You subtract 1 from the length property of an array.
